# 75. Sort Colors
https://leetcode.com/problems/sort-colors/

**Difficulty:** Medium  
**Topic:** Arrays, Dutch National Flag Algorithm

---

## Problem Summary
Given an array `nums` with values 0, 1, and 2 representing colors:

- 0 → Red
- 1 → White
- 2 → Blue

Sort the array **in-place** so that
all 0s come first, then 1s, then 2s.

You must solve without using extra space or built-in sort.
